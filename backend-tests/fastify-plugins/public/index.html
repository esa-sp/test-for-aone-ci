Fastify Example

Modular plugin architecture with encapsulated contexts.

// Register plugins with prefixes
fastify.register(require('./plugins/health'))
fastify.register(require('./plugins/users'), {
  prefix: '/api/users'
})
fastify.register(require('./plugins/products'), {
  prefix: '/api/products'
})

API Endpoints

Each plugin registers its own routes under a prefix. Plugins are encapsulated.
