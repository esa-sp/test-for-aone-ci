NestJS multi-module

NestJS Example

Multi-module architecture with feature modules, guards, and global API prefix.

@Module({ imports: [HealthModule, UsersModule, ProductsModule] })
export class AppModule {}

// main.ts
app.setGlobalPrefix('api')

// users.controller.ts
@Controller('users')
@UseGuards(ApiKeyGuard)
export class UsersController { ... }

API Endpoints

Each feature module has its own controller. Users module is protected by ApiKeyGuard (x-api-key: secret).
