FC Handlers Example

File-based routing for serverless functions — no framework needed, just drop files in /api/

// api/foo.js — Node classic style
module.exports = (req, res) => {
  res.end(JSON.stringify({ from: 'foo' }))
}

// api/health.js — Web fetch style
module.exports = {
  fetch(request) {
    return Response.json({ ok: true })
  }
}

// api/items.js — Named HTTP method style
exports.GET = () => Response.json({ list: [] })
exports.POST = () => Response.json({ created: true }, { status: 201 })

API Endpoints

Three handler styles, all auto-detected from the /api/ directory. Click "Send" to test live.

How It Works

The platform scans your /api/ directory and generates a dispatcher automatically.

📁

File-based Routing

Each file in /api/ becomes a route. api/foo.js → /api/foo

🎭

Multi-style Handlers
